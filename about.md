# AdvancedSumAttempts
Show total attempts counts (and many other stats) for the levels visible on screen, including gauntlets and level lists.

This mod was written entirely using the Nano text editor.

Mod idea suggested by [joar0229](https://discord.com/users/1002209858633867385).
